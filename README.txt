Bumbal connector is a plug-in for sending orders directly from WooCommerce to Bumbal planning software.

== Description ==

With this plug-in you can send orders to Bumbal automatically. Customers can plan them selves on a route after payment.

== Installation ==

1. Upload `bumbal.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set-up the Bumbal plug-in under the WooCommerce settings -> Shipping -> Bumbal
4. Fill in your Bumbal instance with API-key
5. Select the shipping method whereby orders are send to Bumbal


== Frequently Asked Questions ==

= Do I need a Bumbal account for using this plugin? =

Yes, you need Bumbal software for using this plugin.

= Can I install this plug-in myself? =

Yes, just hit 'install' and fill in the required settings.

= Where can I find the settings? =

You can find the settings in the WooCommerce settings menu under shipping. There is a seperate section called 'Bumbal'.

= Why is the plug-in not working? =

Check if you fill in the right API-key and Bumbal instance in the settings. If it's still not working please get in touch with Bumbal support.
